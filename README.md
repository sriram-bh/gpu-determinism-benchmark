# GPU Determinism Benchmark

This project measures how far CUDA GPU kernel scheduling deviates from a formally certified deterministic reference schedule, expressed using the Lingua Franca reactor model. A toy GPT-2 inference pipeline runs on NVIDIA Thor hardware, and each kernel dispatch is captured via CUPTI and compared against a mathematically guaranteed ground truth schedule (G*) derived from Lingua Franca's logical time semantics. The gap between G* and actual GPU execution is quantified by a novel scalar metric called the Deterministic Deviation Index (DDI), which separately tracks order violations (kernels firing in the wrong logical sequence) and timing jitter (kernels firing in the right order but at the wrong wall-clock time). The project is motivated by the industry tension between GPU probabilistic scheduling and LPU-style determinism, and explores a hybrid kernel policy — enforcing deterministic CUDA configurations during prefill and early decode (where numerical divergence is highest, per prior CPU-level simulation) and relaxing to throughput-optimized settings during late decode — as a practical engineering contribution toward closing that gap.
